Progression Open App

Progression is a backing-track tool for musicians. Enter a chord progression, set your key and tempo, and loop it while you practice scales, solos, or chord shapes.

It's easy to get started in seconds — and when you're ready to go deeper, there's a lot to explore.

Practice Ideas

Each example opens the app pre-configured. Press play and start practicing.

Triads!

Play all 3 major triad inversions on a string set around the circle of 4ths. For example, limit yourself to the G, B, and high E strings. Repeat for the other 3 string set combinations!

Try it →

Triad Progression

Play triads over a basic progression mixing major and minor chords. To start, try to stay on one string set and in one area of the neck. This example stays in a single key, but you can manually switch keys or have the player cycle the keys.

Try it →

12-bar blues

Classic I–IV–V form with dominant 7ths at a medium rock tempo. Solo over the changes or comp along. Swap the key to match a jam partner.

Try it →

Mixolydian groove

The I–V–bVII–IV progression is the backbone of classic rock and blues-rock. The bVII chord is what makes it Mixolydian — it implies the flatted 7th, so reach for the Mixolydian scale (major scale with a b7) instead of the regular major scale. Try it in A to match most classic rock jams.

Try it →

Diatonic chords in one key

Loop all seven diatonic chords in order at a slow tempo. Great for learning how each chord sounds within the key. Try soloing with just the major scale throughout.

Playback screen

The main screen shows everything you need while you play.

Preset pill

The pill in the top-left shows the currently loaded preset name. A * means you have unsaved changes. Tap it to save, revert, or switch presets without opening Setup.

Key display

The large key name at the top (e.g. C) shows the current key. Tap it to open a key grid — select a new key and it takes effect at the start of the next loop. In Custom loop mode, the same grid is where you build your key sequence instead.

Chord chips

Each chip shows the resolved chord name for the current key. The active chip highlights during playback, and bar-progress dots appear below it when Bars per chord is greater than 1.

Beat indicator

Four dots below the chord strip pulse on each beat.

Voice-lead

Each chord uses the inversion closest to the previous chord. Flows continuously across all chords and across Cycle laps. Try Voice-lead →

Bars per chord

Sets how many bars each chord plays before the progression advances. Individual chords can override this with per-chord bar notation (V:4). Try 4 bars →

Tempo

Tapping BPM opens a popover with a slider, ± nudge buttons, and a tap-tempo button. Changes apply immediately, even during playback.

Loop modes

Controls how the key changes (or doesn't) across repeated runs. Set via the Loop mode pill on the main screen.

Loop

Repeats the progression indefinitely in the configured key. The default mode.

Cycle 4ths

Plays the progression through all 12 keys ascending by perfect fourths. The key display updates each lap. Try Cycle 4ths →

Cycle 5ths

Same as Cycle 4ths, ascending by perfect fifths. Try Cycle 5ths →

Custom key sequence

Define your own key sequence in any order. Switch the Loop mode pill to Custom, then tap the key display to open the key grid — tapping a key adds it to the sequence, tapping it again removes it. The sequence appears as a numbered list below the grid; reorder with ↑↓ and remove with ×.

Each entry plays one complete run through the progression before advancing to the next key. Accidentals follow the target key's natural spelling.

Try Custom (C→F→Bb→Eb) →

KEY scrubber

When any cycle mode is active, a KEY row appears in the action bar showing every key in the sequence. The current key is highlighted. In Custom mode this row appears as soon as you switch to Custom — even with just one key — so you can see the sequence forming as you build it.

  • While playing (Auto mode) — keys advance automatically after each full run through the progression. Tap a segment to queue a jump to a specific key; it takes effect after the current key's run completes. Tap the same segment again to cancel.
  • While playing (Manual mode) — keys never advance on their own. Tap a segment to queue a key change; it takes effect when the current section completes. Tap again to cancel.
  • While stopped or paused — tap a segment to set the starting key. Chord chips and the key display update immediately.

Play / Pause / Stop

Play / Pause — tap once to start, tap again to pause at the current chord. Spacebar also works when no text input is focused.

Stop (■) — appears next to Play while paused; resets playback to the beginning.

Setup and Mix buttons live in the action bar at the bottom left and right.

Progression syntax

The progression input accepts space-separated tokens. Roman numerals and absolute chord names can be freely mixed.

Type Examples Notes
Major triad I IV V → Uppercase Roman numerals
Minor triad i iv vi → Lowercase Roman numerals
Flat scale degree i bVII bVI → Prefix b lowers by a semitone
Absolute chord I IV Am V → Suffix m for minor; mix freely with Roman numerals
Major 7th Imaj7 IVmaj7 iim7 V7 →
Dominant 7th iim7 V7 I → Uppercase numeral + 7
Minor 7th iim7 V7 Imaj7 → Lowercase numeral + 7
Diminished triad I viidim I → Also °
Augmented triad Iaug IV I → Also I+
Suspended Isus4 I Vsus4 V →
Half-diminished iim7b5 V7 i → Also ø
Fully dim 7th viidim7 I → Also °7
Minor-major 7th imMaj7 iv i →
Dom 7 suspended V7sus4 V7 I →
Per-chord bars I:4 IV:2 V:2 → Append :n to any chord

Repeat a token to extend its duration: I I vi V gives I two iterations at the default bar count. Chord chips show resolved names and highlight during playback.

Sections & arrangement

Build song structure with multiple sections. A single section (the default) just loops.

Sections

Each section has its own progression. Reorder with ↑↓, delete with ×.

Arrangement

An optional sequence of section numbers that controls playback order — 1 means section 1, 2 means section 2, and so on. Leave it empty to play all sections in order.

For example, 1 1 2 3 2 3 plays section 1 twice, then alternates sections 2 and 3 twice. Append :n to repeat a section multiple times: 1:4 2:2 plays section 1 four times then section 2 twice.

Try 3-section song →

Auto / Manual

Controls both section and key advancement.

  • Auto — sections and keys advance automatically. Use the scrubbers to queue jumps ahead of time.
  • Manual — nothing advances without a tap. The progression holds the current section until you tap the SONG scrubber, and holds the current key until you tap the KEY scrubber. Applies to any active cycle mode.

SONG scrubber

Appears in the action bar when the arrangement has two or more steps.

  • While playing — tap a segment to queue a jump to it; takes effect after the current section completes. Tap again to cancel.
  • While stopped or paused — tap a segment to set where playback will start.

Presets

The preset pill in the top-left corner shows the currently loaded preset. Tap it to open the preset menu.

Starter presets

A set of ready-to-play configurations — progression, key, tempo, style, and mix included. Tap any to load it. Use Save as new… to fork it into your own preset.

My presets

Tap + Save preset (or Save as new… from the menu) to name and save the current state. Your presets appear at the top of the menu.

Once a preset is loaded, the pill shows its name. Changing any setting marks it with *. From the menu:

  • Save — overwrite the preset with the current state (enabled when modified)
  • Revert — restore the preset's saved state (enabled when modified)
  • Save as new… — save as a new named preset without affecting the loaded one

Tap the pencil icon (✎) next to a preset to rename it, or × to delete it (with confirmation).

Default preset

Tap the pencil icon (✎) on one of your presets and check Set as default. Opening the app with no link parameters at all then loads that preset instead of the built-in defaults. Only one preset can be default at a time — checking it on another preset replaces the old one. Starter presets can't be set as default directly — use Save as new… to make your own copy first, then mark that as default.

A share link with specific settings still behaves as before — your default preset only applies when there's nothing else to go on.

App settings

Theme

Light / Dark toggle in Setup. Defaults to dark. Stored per-device — not included in URLs or presets.

Keep screen on

Toggle in Setup. When enabled, prevents your screen from dimming during playback.

Share link

Every change you make is reflected in the URL — key, progression, tempo, mix, and more. Use Copy share link in Setup to copy it and share or bookmark your exact setup.

Mix

Open from the Mix button in the action bar. Each instrument — Chords, Bass, Drums — has a mute toggle and volume slider, plus a master volume. All changes apply live during playback.

Looper (Experimental)

Experimental — an early spike to learn what a polished looper should look like, not a finished feature. Expect rough edges and changes.

Enable it with the Looper (Experimental) toggle in Setup. To record:

  1. Set up a single-section progression in Loop mode (no key changes).
  2. Press Record Loop next to Play — if playback isn't already running, this starts it for you. The button label changes to show it's armed.
  3. The progression plays through once in its entirety first, to establish the beat.
  4. The second time through, it records.
  5. Recording stops automatically once that pass completes, and the loop starts playing back alongside the progression.

Mic timing varies by device — sometimes by a lot (one laptop needed roughly 50ms, while a phone needed closer to 350ms) — so the same Setup section has a Loop sync offset field to nudge the recording earlier or later until it lines up. A good way to dial it in: make your first recording just your voice counting each bar out loud (e.g. "1, 2, 3, 4 — 2, 2, 3, 4 — 3, 2, 3, 4 — 4, 2, 3, 4"), then adjust the offset until what you hear matches the beat.

The loop gets its own Looper group in Mix once enabled, with volume/mute plus three optional cleanup controls for cleaning up mic noise — a hi-pass filter, a compressor, and a limiter — each on/off (or, for the compressor, an amount) so you can hear what each one is actually doing to your recording. The loop survives reloading the page — delete and re-record any time from the same button.
